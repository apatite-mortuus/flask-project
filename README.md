# DemCoHub
## Описание
Проект представляет собой сайт с возможностями контроля версий песен/композиций и обмена демозаписями. Пользователь может создать репозиторий, выдать права доступа к нему другим пользователям, загружать файлы в репозиторий, отслеживать версии и перемещаться между ними, скачивать файлы на свой ПК, а также загружать демозаписи в свободный доступ для просмотря другими пользователями. Проект сочетает в себе возможности Git и GitHub, а также социальных сетей.

Описание идеи: проект является удобным инструментом для независимых музыкантов, а также продюсеров и звукорежиссеров. Веб-приложение предоставляет возможность создавать репозитории, добавлять в них других пользователей, загружать и удалять файлы на сервере, переключаться между разными версиями (коммитами), создавать ветки, скачивать файлы с репозитория. Также имеется возможность публикации аудиофайлов и оценки чужих записей.<br>
Описание реализации и технологий: проект основан на базе данных с таблицами, описанными в ТЗ. Все взаимодействие с БД проиходит в обработчиках Flask. Реализованы различные формы с помощью flask-wtf и WTForms. Для взаимодействия клиента с сервером были созданы обработчики POST и DELETE запросов от JavaScript. Реализована также работа с файловой системой при помощи библиотек os, pathlib, shutil, io для сохранения и удаления файлов, создания папок, удаления папок и директорий, отправки файлов и архивов клиенту  на сервер. Необходимые библиотеки для запуска: flask, flask-login, flask_wtf, wtforms, sqlalchemy.<br>
