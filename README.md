# DemCoHub
## Описание
Проект представляет собой сайт с возможностями контроля версий песен/композиций и обмена демозаписями. Пользователь может создать репозиторий, выдать права доступа к нему другим пользователям, загружать файлы в репозиторий, отслеживать версии и перемещаться между ними, скачивать файлы на свой ПК, а также загружать демозаписи в свободный доступ для просмотря другими пользователями. Проект сочетает в себе возможности Git и GitHub, а также социальных сетей.
